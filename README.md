# LifeAI-mobile

Este repositório é sobre um projeto acadêmico realizado na **Escola Tecnológica FPF Tech**.  
Nosso objetivo com este projeto é **aprender profissionalmente** e **propagar a saúde por meio de tecnologias**.

---

## Visão Geral

O **LifeAI Mobile** é um aplicativo Android desenvolvido em **Kotlin**, com o propósito de auxiliar usuários a acompanharem hábitos de saúde e bem-estar, oferecendo insights e recomendações personalizadas baseadas em dados e inteligência artificial.

O projeto faz parte de uma solução integrada, composta por **aplicativo mobile**, **backend** e **infraestrutura em nuvem** hospedada na **AWS (Amazon Web Services)**, utilizando **Kubernetes (K3s)** para orquestração e deploy contínuo.

---

## Objetivos do Projeto

- Desenvolver um aplicativo completo utilizando **Jetpack Compose** e **arquitetura MVVM**.  
- Criar uma API backend segura e funcional em **Django / Node.js**, com integração ao banco de dados **MySQL**.  
- Aplicar conceitos de **DevOps**, **Docker** e **Kubernetes** em ambiente **AWS EC2** real.  
- Promover boas práticas de **programação, versionamento e documentação** no desenvolvimento de software.
